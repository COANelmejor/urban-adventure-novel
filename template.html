COANelmejor's Template

This template include the necesary frameworks and code, to start a "Mobile First" and "Responsive Design" WebSite inside of codepen.io. This configuration includes:

  • A meta viewport tag, that recognize screen width and device.
  • A meta charset tag, with UTF-8 to use any kind of chacracters.
  • Bootstrap
  • jQuery
  • Font-Awesome
  • Animate.css
  • Ubuntu Font from Google Fonts
